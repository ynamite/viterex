# ViteRex für REDAXO 5

Das Addon ist nur ein Teil von ViteRex und nicht separat erhältlich (oder nützlich).
Es bietet ein paar Utility-Funktionen für die Verwendung mit Vite JS in Zusammenhang mit Redaxo.
